1.1주차 배운 내용 정리
vscode, git, vercel 등 개발에 쓰이는 툴들을 설치하였고,  기본적인 사용법에 대해 배웠습니다.

git에 파일을 연동시키는 방법에 대해 배웠습니다.
관련 코드
git status
git add <fillname>
git commit -m "업데이트 내용"
git push

자기소개를 하는 나만의 프로필 페이지를 만들고 실제로 배포해보았습니다. 
<title> : 페이지의 이름을 수정
<h> : 단락의 제목 부여
<p> : 단락 내용 부여
<img src="파일이름" alt="파일설명" width="" height=""> : 이미지파일을 연결하고 크기를 조정
